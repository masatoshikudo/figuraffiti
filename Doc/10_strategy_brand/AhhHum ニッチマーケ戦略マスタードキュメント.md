# AhhHum ニッチマーケ戦略マスタードキュメント

本ドキュメントは、`AhhHum` のブランド思想、ニッチマーケ戦略フレームワーク、認知拡大の運用方針、収益化の順序を一本化したマスタードキュメントである。  
旧 `Figuraffiti` 名義の戦略文書、および `統合戦略` と `デュアル・エンジン戦略` の重複内容を整理し、現時点の意思決定基準として再定義する。

---

## 1. 戦略の結論

AhhHum の戦略は、次の一文に要約される。

> **都市の余白に置かれた小さな違和感を、共鳴する少数者の継続的な探索体験へ変換し、そこから高関与の物理商品と共創権へ段階的に接続する。**

重要なのは、「最初から大衆を狙う」のではなく、**少数の強い共鳴者が熱狂する構造を先に作り、その熱量を可視化することで周辺市場へ広げる**ことにある。

---

## 2. ブランドの核

### 2.1 Why

> **消費されない関係を、都市の余白に置く。**

AhhHum は、家の中で消費されて終わるオブジェでも、短期的なSNSネタでもない。  
都市の風景と探索者の視線のあいだに、繰り返し立ち上がる関係を設計するプロジェクトである。

### 2.2 顧客が本当に片付けたいジョブ

AhhHum のコア顧客が片付けたいジョブは、次の通り。

- 見慣れた日常動線を、自分だけが知る秘密のある場所として再解釈したい
- 単なる消費ではなく、文脈のある発見体験に参加したい
- ストリートカルチャーに触れたいが、危険や違法性ではなく、美学と解釈のレベルで関わりたい
- 説明過剰なエンタメではなく、余白のある世界観に属したい

### 2.2.1 最初に狙うべきビーチヘッド顧客

「共鳴する少数者」とだけ置くと対象が広すぎて、初期施策がぼやける。  
したがって、Phase1 で狙うべき最初の顧客は以下に限定する。

- 都市部で徒歩移動が多い 20-30代
- Instagram / TikTok で街・建築・アート・裏路地系のコンテンツを日常的に見る
- 収集欲よりも「知っている場所が増えること」に快感がある
- 違法性の高いカルチャー当事者ではなく、周辺の美学消費者・観察者である
- 休日に 30-90 分程度の回遊行動が可能

つまり最初の主戦場は「全国」ではなく、**都内23区を初期市場としつつ、その中でも徒歩回遊が成立し、少数スポットでも密度感が演出できる重点区から始める** べきである。

### 2.3 守るべきブランド態度

- **Anonymous**: 作り手の自己顕示より、都市と物体を主役にする
- **Silent**: 説明しすぎない
- **Contextual**: 場所の歴史や地形との関係を重視する
- **No Cheapening**: 値引きや乱売で世界観を崩さない

## 3. ニッチマーケ戦略フレームワーク 5+1 ステップ

### Step 0: 戦略基盤の構築

最初に戦う場所は、**TikTok / Instagram Reels** と **Webマップ** である。

- `TikTok / Reels`: 認知の獲得。都市の風景がズレる瞬間を短尺で見せる
- `Webマップ`: 共鳴者の受け皿。曖昧サークル、Last Seen、ティッカーで「生きている探索系」を成立させる
- `NFC`: 発見を「見た」から「自分が動かした」に変換する物理トリガー

ここでの前提は、**iPhoneでも動く導線が必要**ということ。したがって NFC は Web NFC ではなく、**NFCタグに URL を埋めて Safari を起動し、既存 Web に接続する方式**を基本とする。

また、媒体ごとの役割を混同しない。

| 媒体 | 主目的 | 見せるもの | 成功指標 |
| :--- | :--- | :--- | :--- |
| TikTok / Reels | 新規認知 | 都市の風景がズレる瞬間、発見の一瞬、違和感の映像 | 視聴完了率、プロフィール遷移率 |
| Instagram | 世界観の定着 | 写真、設置文脈、静かなアーカイブ | 保存率、プロフィール回遊率 |
| Webマップ | 行動転換 | 曖昧サークル、Last Seen、ティッカー、発見導線 | 詳細タップ率、現地遷移率、NFCタッチ率 |
| X | 補助的拡散 | 発見報告、更新、考察、リアルタイム性 | 言及数、再訪流入 |

### Step 1: ジョブ定義とプロダクトの再翻訳

AhhHum は「フィギュア販売」ではなく、次のように再翻訳する。

- 悪い定義: 小さなアートフィギュアを設置・販売するサービス
- 良い定義: **都市に隠された文脈を発見し、自分の行動で痕跡を更新できる探索体験**

そのため、Phase1 の主役は物販ではなく、以下の 3 点になる。

- 曖昧なサークル
- Last Seen
- NFC による発見記録

### Step 2: キャズム越えの市場エントリー

いきなり「現地に行け」は重い。  
そのため、入口は以下の 2 段構えにする。

1. **低カロリー入口**  
   動画視聴、場所予想、ティッカー閲覧、マップ上の曖昧サークル確認

2. **高関与入口**  
   実際に現地へ行き、NFC をタッチして自分の行動で Last Seen を更新する

つまり、AhhHum のキャズム越えは「安い小物」ではなく、**認知コストの低い観察体験から、現地行動を伴う高関与体験へ段階移行させること**で行う。

ここで重要なのは、**市場エントリーの入口** と **収益化の入口** を混同しないこと。  
Phase1 の市場エントリーは「無料で覗けること」が正しく、Phase2 以降の収益化ではじめて物理キーや観賞用商品が意味を持つ。

したがって、キャズム越えの設計は次の 2 軸で考える。

1. **参加の入口**  
   見るだけ -> 気になる -> 現地に行く -> 見つける

2. **課金の入口**  
   無料参加 -> 記念として持つ -> 機能付き物理キーを買う -> 設置権を買う

### Step 2.5: 密度戦略

この事業は、スポットが散発的だと「地図に何もない」「行っても成果が出ない」という印象を生みやすい。  
そのため、初期市場は **都内23区** としつつ、実際の運用は **23区内の重点区から順次広げる密度戦略** を取る。

- 市場定義はまず `都内23区`
- ただし初期配置は、徒歩 30-45 分で複数探索できる重点区に集中させる
- 23区内で稼働実績が出た後、隣接区へ段階拡張する
- SNS露出量ではなく、各区での「再発見可能性」と「再訪理由」を優先する
- 関東エリア展開は、23区内で密度と運用の再現性が確認できてから行う

地理展開の順序:

1. **Phase 1a: 都内23区内の重点区集中**
   例: 渋谷区、新宿区、世田谷区、目黒区など、徒歩回遊とカルチャー文脈が成立しやすい区から始める

2. **Phase 1b: 都内23区内の横展開**
   重点区での再訪率、発見記録、運用安定が確認できたら、他区へ拡張する

3. **Phase 2以降: 関東エリア展開**
   東京で成立した密度設計と運用ルールをもとに、神奈川、千葉、埼玉へ広げる

### Step 3: 大きな波の設計とブランド毀損リスク管理

AhhHum はバズを否定しないが、バズはあくまで入口である。  
運営上の基本方針は次の通り。

- バズ要素は `覆面アーティスト` 側に寄せる
- ブランド本体 `AhhHum` 側は静かに文脈だけを提示する
- 派手な露出よりも「その場所である理由」を優先する
- 無秩序な設置、過度な説明、安売り、雑な量産は避ける

また、盗難・破壊・消失は失敗ではなく、**都市に置く以上必ず起きる現象**として扱う。  
将来的には「消失」をアーカイブと伝説表示に変換し、負債ではなく物語資産にする。

ただし、ここには運用・法務・安全面の前提条件がある。  
美学だけで押し切ると、初期の炎上や撤去でブランドが毀損する。

最低限のルール:

- 私有地・営業施設・公共交通インフラなど、トラブルが大きい場所は原則避ける
- 危険行為を誘発する場所、高所、車道近接は避ける
- 撤去依頼や苦情が来た場合の対応ポリシーを持つ
- 「違法設置を推奨するブランド」と誤認されない表現ルールを持つ
- ユーザー設置解放前に、設置審査基準を文章化する

### Step 3.5: デュアル・エンジンの接続ルール

現状の二層構造は有効だが、人格とブランドの境界が曖昧だと混乱が起きる。  
したがって両者の関係は次のように定義する。

- 覆面アーティストは「話題化装置」
- AhhHum は「意味の保存装置」
- 話題投稿は個人側で行い、体験参加の受け皿だけを AhhHum に集約する
- ブランド側では、個人の熱量を再演せず、体験と文脈の整理に徹する
- 片方が停止しても、もう片方の価値が残るよう依存度を下げる

### Step 4: バリューラダーによる顧客育成と収益化

AhhHum の価値の階段は、以下の順で設計する。

1. **無料**  
   SNS動画、マップ閲覧、ティッカー閲覧、曖昧サークル探索

2. **高関与無料体験**  
   現地での発見、NFCタッチ、Last Seen 更新、発見ログへの参加

3. **中間商品**  
   自宅観賞用フィギュア、物理キー、限定小物

4. **主力商品**  
   設置用フィギュア、設置権、共創権、文脈設計を伴う特別商品

5. **上位商品 / B2B・IP拡張**  
   コラボ展示、アパレル、都市回遊企画、ブランドコラボ

ポイントは、**先に体験の熱量を作り、その後に物販を接続する**こと。  
物販先行では、AhhHum の魅力は単なるトイブランドに縮退する。

補足すると、低価格商品は「誰でも買える量販入口」ではなく、**世界観を壊さずに関与を深めるための記念物・鍵商品** として扱う。  
この定義を外すと、Step2 の無料入口戦略と矛盾する。

### Step 5: 熱狂維持と模倣対策

模倣されにくい本質は造形そのものではなく、以下にある。

- **文脈の一致**: なぜその場所なのか
- **発見ログの蓄積**: 誰かが実際に見つけている事実
- **匿名的で静かな美学**: 語りすぎない態度
- **共創の審査性**: 設置理由に対するリテラシーの壁

したがって模倣対策は法務だけでなく、**文脈選定能力そのものを参入障壁化すること**で行う。

さらに、参入障壁は 1 つでは弱い。  
AhhHum は以下の複合モートで守るべきである。

1. **場所モート**  
   どこに置くかの解像度

2. **ログモート**  
   発見履歴、Last Seen、再発見頻度という蓄積データ

3. **審査モート**  
   共創申請を通すブランド基準

4. **世界観モート**  
   語り口、匿名性、静けさの一貫性

5. **コミュニティモート**  
   少数の高関与者が「分かる人には分かる」を維持すること

## 4. デュアル・エンジン運用方針

ニッチマーケ戦略を実務に落とす際は、次の 2 エンジンを分ける。

| エンジン | 役割 | 主戦場 | トーン |
| :--- | :--- | :--- | :--- |
| 覆面アーティスト | 認知獲得、事件化、制作の狂気を見せる | TikTok / Reels / X | 人間味、偏愛、事件性 |
| AhhHum | 世界観維持、探索導線、長期価値の受け皿 | Webマップ / Instagram / 将来アプリ | 静寂、匿名、文脈 |

運用ルール:

- 覆面アーティストは「入口」
- AhhHum は「帰着点」
- 事件性は個人側、静かな価値はブランド側に残す
- どちらも相互送客するが、トーンは混ぜない

## 5. 現時点のプロダクト戦略

### Phase 1

目的は、**認知とシステムの稼働確認** である。

最小要件:

- 曖昧サークル表示
- Last Seen 表示
- グローバル・ティッカー
- ログイン導線
- NFCタッチによる発見記録

地理方針:

- 初期市場は都内23区
- ただし実運用は 23区内の重点区に集中して開始する
- 「23区対応」と「23区全域に均等配置」は別物として扱う

現時点では、NFC は `URL埋め込みタグ -> /discover/nfc -> 発見記録` の方式を基本とする。  
QR は補助導線として残してよいが、主体ではない。

Phase1 の完了条件:

- 特定エリアで継続的にティッカーが発生している
- 初回訪問だけでなく、再訪ユーザーが一定割合いる
- 発見記録が「たまたま」ではなく、週次で再現している
- 苦情・撤去・誤作動への運用が回っている
- 重点区から隣接区へ広げても、密度感が崩れない

### Phase 2

目的は、**収益化の開始**。

- 物理キー用フィギュア
- 休眠スポット / 伝説表示
- マップ上での差別化体験
- 世界観に沿った限定商品

地理方針:

- 都内23区内での拡張を継続
- 23区モデルをテンプレ化し、関東近郊へ移植可能な形にする

Phase2 に進む条件:

- Phase1 の探索体験が単独でリテンションを生んでいる
- 「欲しい」の理由が造形だけでなく、体験拡張として説明できる
- エリア密度が低すぎず、物理キーが機能するだけの休眠アーカイブがある
- 23区内での展開ルールが標準化され、他エリアへ複製可能になっている

### Phase 3

目的は、**共創の解放**。

- 設置申請
- 文脈審査
- アンバサダー制度
- ユーザーが都市に意味を配置する権利の販売

Phase3 に進む条件:

- 運営だけでなく、ユーザー側にも設置したい動機が明確に生まれている
- 審査基準が言語化され、ブランド毀損を抑えられる
- 既存コミュニティに、模倣ではなく文脈を評価する文化がある

### Phase 4

目的は、**IP展開**。

- アパレル
- 都市回遊企画
- コラボレーション
- 海外展開

地理方針:

- 関東エリアでの成功を足場に、他都市圏へ広げる
- 地域拡張ごとに、東京の文脈を複製するのではなく、その土地ごとの文脈へ翻訳する

Phase4 に進む条件:

- AhhHum 単体で「何のブランドか」が説明可能
- 商品単体ではなく、探索体験ごと想起される状態になっている
- 他ブランドが借りたいのが造形だけでなく、世界観と参加構造になっている

## 6. KPIの持ち方

初期は売上より、以下を優先指標とする。

- サークル閲覧数
- サークル詳細タップ率
- NFCタッチ数
- 発見記録成功率
- 重複記録率
- ティッカー発生件数 / 日
- 1スポットあたりの再発見頻度
- SNS動画からマップへの流入数

ただし、閲覧系指標だけでは「面白そう」で止まっているのか、「実際に体験へ移行している」のかが見えない。  
そのため KPI は、最低でも次の 4 層で持つ。

1. **認知**
   SNS再生数、保存率、プロフィール遷移率、マップ流入数

2. **行動転換**
   サークル詳細タップ率、現地訪問意向率、NFCタッチ率、発見記録成功率

3. **継続**
   7日 / 30日再訪率、再発見者比率、エリア内回遊数

4. **供給健全性**
   アクティブスポット数、スポット生存日数、苦情件数、撤去率、誤検知率

また、`重複記録率` は高いほど良いのか悪いのかが曖昧なので、**再発見率** と **不正・誤作動率** に分解して扱うほうがよい。

Phase2以降に追加する指標:

- 物理キー購入率
- 継続訪問率
- コアユーザー比率
- 共創申請数

## 7. 意思決定ルール

今後の判断基準は、次の通り。

### やること

- 日常動線上で成立する探索体験を優先する
- 最初の勝ち筋を特定エリアで先に作る
- 発見の痕跡が残る仕組みを強化する
- NFC を「自分が世界を更新した」感覚の核に置く
- 物販は体験の後に置く
- 説明より文脈を設計する

### やらないこと

- 全国同時展開を前提にすること
- 量だけを追う安易な設置拡大
- 世界観と無関係なバズ施策
- 値引き主導の販売
- 物販先行でブランドを理解させようとすること
- iPhoneで使えない NFC 実装を前提にすること
- 法務・安全ルールを曖昧なまま UGC 設置を解放すること

## 9. 結論

AhhHum は、「ストリート風の物販」ではなく、**都市の余白を探索可能なメディアへ変えるニッチ体験ブランド**として戦うべきである。  
戦略の中核は、`認知 -> 共鳴 -> 現地行動 -> 発見記録 -> 物理キー -> 共創` の順序を崩さないことにある。

つまり、AhhHum の勝ち筋は次の 3 点に集約される。

- **少数の強い共鳴者から始める**
- **NFC を体験の核として運用する**
- **物販ではなく探索体験をブランドの本体に置く**

# BENTO型ヘッダーデザイン案

## コンセプト

PC画面のヘッダーをBENTO型（グリッドレイアウト + カード型要素）に変更し、よりコンパクトでモダンなデザインにします。

## デザイン案

### レイアウト構造

```
┌─────────────────────────────────────────────────────────────┐
│  [Logo Card]  [遊び方]  [ガイドライン]  [User]  [マップを見る] │
└─────────────────────────────────────────────────────────────┘
```

### 各要素のデザイン

#### 1. ロゴカード
- **サイズ**: コンパクト（高さ: 48px）
- **スタイル**: 角丸カード、背景色あり
- **内容**: アイコン + テキスト（またはアイコンのみ）

#### 2. ナビゲーションカード
- **サイズ**: 各リンクを個別のカード型ボタンに
- **スタイル**: ホバー時に背景色が変わる、角丸
- **内容**: 「遊び方」「ガイドライン」をそれぞれ独立したカードに

#### 3. ユーザーメニューカード
- **サイズ**: コンパクトなアイコンボタン
- **スタイル**: 角丸カード、ホバー時に背景色が変わる

#### 4. アクションボタンカード
- **サイズ**: コンパクトなボタン
- **スタイル**: プライマリカラー、角丸

### 全体の仕様

- **高さ**: `h-12` (48px) - 現在の `h-16` (64px) から削減
- **グリッドレイアウト**: `grid-cols-auto` で自動配置
- **ギャップ**: `gap-2` (8px)
- **パディング**: `px-4` (16px)
- **背景**: 半透明 + ぼかし効果（現状維持）

### レスポンシブ対応

- **モバイル**: 現状のデザインを維持（BENTO型はPCのみ）
- **タブレット以上**: BENTO型レイアウトを適用

## 実装のポイント

1. **グリッドレイアウト**: `grid` と `grid-flow-col` を使用
2. **カード型要素**: 各要素に `rounded-lg` と `p-2` または `p-3` を適用
3. **ホバーエフェクト**: `hover:bg-accent` などでインタラクティブに
4. **コンパクト化**: フォントサイズとパディングを調整

## 視覚的な効果

- ✅ よりモダンで洗練された見た目
- ✅ 各要素が独立して認識しやすい
- ✅ コンパクトで画面スペースを有効活用
- ✅ ホバー時のフィードバックが明確
